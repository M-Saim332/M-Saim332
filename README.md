### 🏥 Clinic Management System — Desktop SaaS Product
> A cross-platform clinic management desktop application built with C# and Avalonia UI, currently in active development as a SaaS offering for clinics.

- 🖥️ Built with **Avalonia UI (MVVM)** for a modern, cross-platform desktop experience
- 🗄️ **SQL Server** backend for patient records, appointments, and billing data
- 🔐 Password reset & secure admin account management flows
- 📦 Packaged with an **Inno Setup installer** for easy client deployment
- 🏗️ Clean layered architecture (`Core`, `Data`, `UI`) for maintainability

**Stack:** `C#` `Avalonia UI` `MVVM` `SQL Server (T-SQL)` `Inno Setup`

**Repo:** [github.com/M-Saim332/Clinic-Software](https://github.com/M-Saim332/Clinic-Software)

### 🏫 GoCampusFlow — Multi-Tenant School Management SaaS
> A complete multi-tenant school management SaaS for principals, teachers, student-management staff, and administrators, currently in active development.

- 🏢 **Multi-school tenant isolation** using Supabase Row Level Security (RLS)
- 👥 Role-based dashboards for principals, admins, staff & teachers
- 🧑‍🎓 Student management with attendance workflows, guardian details & soft archival
- 📊 Reports for attendance, enrollment & activity logs with CSV export
- 💰 Finance module & Pakistan-specific ERP enhancements
- 🔐 Supabase Auth with protected routes & role-aware redirects

**Stack:** `Next.js` `TypeScript` `Supabase (Auth/Postgres/RLS)` `Tailwind CSS` `Zod` `React Hook Form` `Recharts`

**Repo:** [github.com/M-Saim332/School-MS](https://github.com/M-Saim332/School-MS)
